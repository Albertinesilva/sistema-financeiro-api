# 💰 Financial System API

Backend API for a financial management system developed with Spring Boot, focused on transaction control, data persistence and application architecture.

---

## 🚀 Objective

This project aims to implement a backend system capable of managing financial data such as transactions, categories and business rules, demonstrating backend development best practices.

---

## 🛠️ Technologies

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 / MySQL (depending on configuration)
- Maven

---

## 📚 Features

- 💸 Financial transaction management
- 📂 Categorization of expenses and income
- 🔄 CRUD operations
- 🗄️ Data persistence with JPA
- 🧠 Business rule implementation
- 🌐 RESTful API

---

## 📂 Project Structure

- `entities/` → domain model
- `repositories/` → data access layer
- `services/` → business logic
- `controllers/` → REST endpoints

---

## ▶️ Running the project

1. Clone the repository:

```bash
git clone https://github.com/Albertinesilva/sistema-financeiro-api
```

2. Navigate to the project folder:

```bash
cd sistema-financeiro-api
```

3. Run the application:

```bash
./mvnw spring-boot:run
```

Or run it via your IDE.
